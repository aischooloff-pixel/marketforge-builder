

## План: Попап «TEMKA NEWS» в стиле Win95

Один компонент — всплывающее окно в стиле Win95 с продвижением Telegram-канала.

### Логика показа
- Счётчик сессий в `localStorage` (`temka_session_count`)
- Показ на каждой 5-й сессии
- Если нажал «Подписаться» — флаг `temka_subscribed`, больше не показывать
- «Позже» — просто закрывает, покажется через 5 сессий снова

### Внешний вид

```text
┌─[📡 TEMKA NEWS]──────── ● LIVE ─┐
│                                  │
│  ✈ Подпишись на канал —          │
│  узнавай о завозах первым        │
│                                  │
│         [ Подписаться ]          │
│           [ Позже ]              │
└──────────────────────────────────┘
```

- Win95-окно (`bevel-raised`, заголовок с `bevel-sunken`)
- Пульсирующая зелёная точка «● LIVE» в заголовке
- Кнопка «Подписаться» → `https://t.me/TemkaStoreNews`, primary
- Кнопка «Позже» → outline, закрывает окно
- Overlay затемнение как у Dialog
